# Cinema 🎬

A professional streaming platform dashboard built with modern JavaScript (ES Modules) and premium CSS.

## 🚀 Features
- **Modern File Structure**: Separated concerns (API, UI, Main).
- **IMDb API Integration**: Fetches trending and top-rated content dynamically.
- **Vidking Player**: Integrated streaming for movies and series.
- **Premium Aesthetics**: Dark purple "Midnight" theme with glassmorphism.

## 📂 Project Structure
- `/assets/css/main.css`: Global styles and design system.
- `/assets/js/api.js`: Handles all communication with external APIs.
- `/assets/js/ui.js`: Manages DOM rendering and UI components.
- `/assets/js/main.js`: Entry point that initializes the application.
- `index.html`: Main entry point.

## 🛠️ APIs Used
- **IMDb Data**: [imdbapi.dev](https://imdbapi.dev)
- **Video Player**: [vidking.net](https://vidking.net)

## 🖥️ Getting Started
Simply open `index.html` in any modern browser (like Opera GX).
